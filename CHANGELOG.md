# Changelog

## [0.1.0] - 2025-01-26

### Added
- Multi-agent orchestration with intelligent routing
- Channel support: CLI, REST API, Telegram, Slack
- 10 specialized agents (personal assistant, coder, marketing, sales, etc.)
- Tool system with built-in tools (calculator, datetime, projects, browser)
- SQLite persistence for sessions and conversations
- Semantic memory with vector search
- Web dashboard for observability
- Cost tracking and budget limits
- Docker deployment support
